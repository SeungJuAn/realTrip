# PRD: 여행 일정 플래너 (Travel Planner)

## 1. 개요

### 1.1 프로젝트 소개
사용자가 여행 일정을 계획하고, 여행지 정보를 탐색하며, 친구들과 함께 일정을 공유/협업할 수 있는 웹 애플리케이션

### 1.2 기술 스택
- **Frontend**: React
- **Backend**: Spring Boot
- **Database**: TBD (PostgreSQL 권장)

### 1.3 대상 사용자
- 국내/해외 여행을 계획하는 개인 및 그룹
- 여행 일정을 체계적으로 관리하고 싶은 사용자

---

## 2. 핵심 기능

### 2.1 수동 일정 편집
**우선순위: 높음**

- 드래그 앤 드롭으로 일정 순서 변경
- 날짜별 일정 타임라인 뷰
- 장소, 시간, 메모 입력
- 일정 복사/붙여넣기
- 이동 시간 자동 계산 (선택)

### 2.2 여행지 정보 제공
**우선순위: 높음**

- 관광지, 맛집, 숙소 검색
- 장소 상세 정보 (주소, 영업시간, 리뷰)
- 지도 연동 (Google Maps / Kakao Maps)
- 북마크/저장 기능
- 카테고리별 필터링

### 2.3 일정 공유/협업
**우선순위: 높음**

- 일정 링크 공유
- 실시간 공동 편집
- 참여자 권한 관리 (보기/편집)
- 댓글 및 투표 기능
- 변경 알림

## 5. API 설계 (초안)

### 5.1 여행 (Trip)
- `POST /api/trips` - 여행 생성
- `GET /api/trips` - 내 여행 목록
- `GET /api/trips/{id}` - 여행 상세
- `PUT /api/trips/{id}` - 여행 수정
- `DELETE /api/trips/{id}` - 여행 삭제

### 5.2 일정 (Schedule)
- `POST /api/trips/{tripId}/schedules` - 일정 추가
- `GET /api/trips/{tripId}/schedules` - 일정 목록
- `PUT /api/schedules/{id}` - 일정 수정
- `DELETE /api/schedules/{id}` - 일정 삭제
- `PUT /api/schedules/reorder` - 일정 순서 변경

### 5.3 장소 (Place)
- `GET /api/places/search` - 장소 검색
- `GET /api/places/{id}` - 장소 상세
- `POST /api/places/bookmark` - 북마크 추가

### 5.4 공유 (Share)
- `POST /api/trips/{id}/share` - 공유 링크 생성
- `POST /api/trips/{id}/collaborators` - 협업자 추가
- `DELETE /api/trips/{id}/collaborators/{userId}` - 협업자 제거

## 6. 데이터 모델 (초안)

### User
- id, email, password, nickname, profileImage, createdAt

### Trip
- id, title, description, startDate, endDate, ownerId, createdAt, updatedAt

### Schedule
- id, tripId, placeId, date, startTime, endTime, memo, orderIndex

### Place
- id, name, address, latitude, longitude, category, imageUrl, rating

### Collaborator
- id, tripId, userId, permission (VIEW/EDIT), invitedAt
